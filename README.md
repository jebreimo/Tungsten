# Tungsten

Library for making OpenGL applications with SDL.

Tungsten is a static C++20 library that provides exception-safe wrappers over
OpenGL objects, a shader/material/lighting layer, text rendering, and an SDL3
application scaffold. It targets both desktop (GLEW + OpenGL + SDL3 + Freetype)
and the web via Emscripten/WebGL.
